MCP Connectors To Build

Keep it minimal initially.

Connector 1 — Filesystem MCP

Expose:

list files
read file
search code
retrieve docs

This alone is powerful.

Connector 2 — Git MCP

Expose:

recent commits
changed files
branch diff
blame info
Connector 3 — Terminal MCP

Expose controlled commands:

pytest
ls
grep
docker ps

Add allowlists for safety.
